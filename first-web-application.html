Building My First Web Application: A Dungeons & Dragons Spellbook

After dedicating hours a day to following the 100 Days of Code Python course on Udemy (shoutout Angela Yu!), I've finally created my very own website using all I've learned in the past few months.

December 22, 2023 Jake Stephens

After months of dedicating hours each day to the "100 Days of Code Python" course on Udemy, guided by the incredible Angela Yu, I've finally reached a milestone that I'm truly proud ofâ€”I've created my very own web application. This project represents the culmination of countless hours of learning and practice, and it's a tangible reflection of how far I've come in my coding journey.

The application I've built leverages my newfound knowledge of RESTful APIs and the Flask framework, both of which were central to the course. I chose to center the project around one of my personal interests: Dungeons & Dragons (D&D). Using a D&D API, I've created an application that allows users to explore the vast array of spells within the game. In its current form, the application offers a user-friendly interface where users can browse through a complete list of spells or search for specific ones. Each spell entry provides detailed information, including its description, range, damage, and other essential stats that any adventurer would need.

While the application is still in its early stages, I'm excited about the possibilities for future updates. One of my primary goals is to expand the functionality by adding pages that categorize spells by class, making it even easier for users to find spells tailored to their characters. This project has not only been a valuable learning experience but also a passion project that has kept me motivated and excited about what I can achieve as I continue to develop my skills as a software developer. I can't wait to see where this journey takes me next!
