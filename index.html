Building My First Web Application: A Dungeons & Dragons Spellbook

After dedicating hours a day to following the 100 Days of Code Python course on Udemy (shoutout Angela Yu!), I've finally created my very own website using all I've learned in the past few months.

December 22, 2023 Jake Stephens

I've finally decided to take my newfound knoweledge of RESTful APIs and the Flask framework to create my own web application. The application uses a Dungeons and Dragons (D&D) API to get all of the spells that're in D&D. It's in the early stages, but right now the user is able to view a list of all spells and/or search for any specific spell and view more information, such as the description, range, damage etc.... In future updates, I hope to add pages to show all spells for each class.
